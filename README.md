## 🌟 Overview

Goodweeks Cam is a high-performance AI tool designed for the creative media industry. Whether you're an artist animating custom characters, a content creator looking for unique visuals, or a professional exploring new digital identities, Goodweeks Cam provides the tools you need with a modern, intuitive interface.

### Key Features
- **Real-time Transformation:** Apply face swaps instantly during live streams or video calls.
- **High Fidelity:** Professional-grade enhancement for lifelike results.
- **Multi-Face Support:** Map different identities to multiple subjects simultaneously.
- **Mouth Masking:** Retain natural mouth movements for realistic speech.
- **Privacy First:** Built-in ethical safeguards and content filtering.

## 🚀 Quick Start

1. **Select Identity:** Choose a source image for the face you want to use.
2. **Choose Target:** Select your webcam or a target video file.
3. **Go Live:** Press the "Live" button and start your transformation!

## 🛠️ Installation

### Prerequisites
- Python 3.11
- FFmpeg
- NVIDIA GPU (Recommended for best performance)

### Setup
```bash
# Clone the repository
git clone https://github.com/ashokmotiji/Goodweeks-Cam.git
cd Goodweeks-Cam

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Running the App
```bash
python run.py
```
